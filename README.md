# NewsPulse: Automated Web Scraper

A high-performance Python-based web scraping engine designed to extract, clean, and structure data from live news feeds. This project demonstrates expertise in **Data Extraction**, **HTML Parsing**, and **Automated Data Persistence**.

## Key Features
- **Dynamic Extraction:** Targets and extracts top trending headlines and source URLs using `BeautifulSoup4`.
- **Structured Storage:** Automatically formats and saves scraped data into a timestamped **CSV file** for easy analysis.
- **Request Management:** Implements custom `User-Agent` headers to simulate browser behavior and ensure reliable connectivity.
- **Data Integrity:** Includes filtering logic to capture only high-relevance content from complex DOM structures.

## Tech Stack
- **Language:** Python 3.x
- **Libraries:** BeautifulSoup4 (HTML Parsing), Requests (HTTP Client), CSV (Data Formatting)
- **Data Output:** Tabular CSV format
